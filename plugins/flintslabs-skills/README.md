# FlintsLabs Skills

Shared Codex skills for FlintsLabs workflows.

This plugin is intentionally skills-only for now. Add MCP servers, apps, or assets later only when there is a concrete need.

## Skills

- `flintslabs-debug`: root-cause debugging workflow.
- `flintslabs-repo-research`: read-only repository research and review workflow.
- `flintslabs-skill-authoring`: create and maintain focused reusable skills.
- `flintslabs-codex-marketplace`: maintain this Codex marketplace and plugin update flow.
- `flintslabs-delivery-check`: final verification and delivery summary workflow.
